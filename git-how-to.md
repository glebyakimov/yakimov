# Инструкция по работе с SSH и GitHub

## 1. Создание SSH-ключа

### Генерация ключа
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
(или -t rsa -b 4096 для RSA)

Где:
- -t - тип ключа (рекомендуется ed25519)
- -C - комментарий (обычно email)

### Дополнительные опции:
- Нажмите Enter для сохранения в стандартную папку ~/.ssh/
- Введите пароль (опционально, но рекомендуется)
- После генерации появятся 2 файла:
  - id_ed25519 - приватный ключ (никому не показывать!)
  - id_ed25519.pub - публичный ключ

## 2. Добавление SSH-ключа в GitHub

1. Скопируйте публичный ключ:
Bash
cat ~/.ssh/id_ed25519.pub | pbcopy
(или откройте файл и скопируйте вручную)

2. На GitHub:
   - Перейдите в Settings → SSH and GPG keys
   - Нажмите "New SSH key"
   - Вставьте ключ в поле "Key"
   - Добавьте понятное название (например, "Work Laptop")
   - Сохраните

## 3. Клонирование репозитория

### Через SSH (рекомендуется):
Bash
git clone git@github.com:username/repository.git

### Проверка подключения:
Bash
ssh -T git@github.com
(должно появиться приветствие с вашим именем на GitHub)
